MAY DAY PHASE FOUR PATCH

Surgical scope only.

Files included:
- src/data/maydayContent.js
- src/data/huntData.js

What this patch does:
- replaces placeholder site links with the real donation, linktree, shop, and vendor or sponsor links in site data
- locks in the confirmed venue, address, and contact email in site data
- replaces the basic hunt placeholder routes with first pass route text based on the uploaded Activities, Vendors, Indoors, and Outdoors sections
- keeps certain hunt stops intentionally marked as onsite only or volunteer gated because those codes are not supposed to exist publicly online

What this patch does not do:
- final world map
- final building specific location tuning
- QR code generation
- redesign
